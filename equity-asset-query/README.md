# 股权资产管理 · 联查校验平台

独立的网页查询工具，支持四表联查（台账、项目对赌、股权历史、股权月度），提供数据校验、导入导出、比例计算等高级功能。

## 功能

- **四表联查**：台账表、项目对赌表、股权历史表、股权月度表
- **查询条件**：项目名称（模糊）、项目编号（精确）、天九持股主体（模糊）、天九持股主体统一社会信用代码（精确）
- **分页 / 排序 / 列显隐**
- **导入导出**：支持台账、天九持股主体、股权历史三张表的 xlsx 导入导出
- **高级功能**：
  - 股权登记 / 退回 / 处置比例计算（自动校验月度表数据）
  - 项目对赌计算（期末 / 时点校验）
- **辅助工具**：环境切换（测试 / 生产）、计算器、天气、后端日志实时查看

## 技术栈

- **后端**：Flask + Blueprint（代理调用表单引擎接口，统一鉴权）
- **前端**：原生 HTML / CSS / JS（深色主题操作面板）

## 目录结构

```
equity-asset-query/
├── app.py                          # Flask 主入口：基础路由（查询/登录/环境/日志）
├── shared.py                       # 共享状态：环境变量、token 缓存、登录逻辑
├── config.py                       # 配置：接口地址、表单ID、字段映射、展示列
├── utils.py                        # 工具函数：xlsx 格式转换、日期解析、数值解析
├── routes/
│   ├── __init__.py
│   ├── import_export.py            # 导入导出路由（9个接口）
│   └── advanced.py                 # 高级功能路由（股权登记计算、对赌计算）
├── templates/
│   └── index.html                  # 前端页面模板（Jinja2）
├── static/
│   ├── css/style.css               # 全局样式（深色主题）
│   ├── js/
│   │   ├── app.js                  # 核心逻辑：表格渲染、查询、分页、排序、登录
│   │   ├── advanced.js             # 高级功能：股权登记计算、项目对赌计算
│   │   ├── widgets.js              # 辅助组件：计算器、天气、日志、环境切换、搜索历史
│   │   └── import.js               # 导入功能
│   ├── 股权变更台账辅助表.xlsx       # 台账导入/导出模板
│   ├── 天九持股主体公司管理.xlsx     # 天九持股主体导入/导出模板
│   └── 股权资产管理历史数据（...）.xlsx  # 股权历史导入/导出模板
├── requirements.txt                # Python 依赖
└── start.command                   # macOS 一键启动脚本
```

## 运行

### 方式一：一键启动（推荐）

双击 `start.command`，脚本会自动：检查/安装依赖 → 启动服务 → 打开浏览器（`http://127.0.0.1:5001`）。

### 方式二：命令行启动

```bash
cd equity-asset-query
pip install -r requirements.txt
python3 app.py
# 浏览器打开 http://127.0.0.1:5001
```

## 模块说明

| 文件 | 职责 |
|------|------|
| `app.py` | Flask 应用入口，注册 Blueprint，包含首页、查询、登录、环境切换等基础路由 |
| `shared.py` | 存放跨模块共享的运行时状态（当前环境、token 缓存、日志队列、登录函数） |
| `config.py` | 所有配置项：环境地址、登录参数、四张表的表单ID/字段映射/展示列/排序/导入模板列映射 |
| `utils.py` | 通用工具函数（xlsx inlineStr→sharedStrings 转换、失败文件解析、日期和数值安全解析） |
| `routes/import_export.py` | 台账/天九持股主体/股权历史的上传、执行导入、导出路由 |
| `routes/advanced.py` | 高级计算路由：股权登记比例计算、对赌前置判断、对赌计算 |
| `static/js/app.js` | 前端核心：表格渲染、四表查询、分页翻页、排序、列显隐、登录、导出 |
| `static/js/advanced.js` | 前端高级功能弹窗逻辑：股权登记计算 + 项目对赌计算 |
| `static/js/widgets.js` | 前端辅助组件：计算器、天气展示、后端日志面板、环境切换、更新日志、搜索历史 |
| `static/js/import.js` | 前端导入功能：文件选择、上传、执行、进度展示 |
| `static/css/style.css` | 全局 CSS 样式（深色主题、表格冻结列、弹窗、高亮等） |
| `templates/index.html` | Jinja2 页面模板，定义 HTML 结构和布局 |
| `start.command` | macOS 双击启动脚本 |
